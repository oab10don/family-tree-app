# 縁図 Enzu

**人と人の関係を、1枚に。** — 医療から相続まで使える関係図（ジェノグラム）ツール。

家族・親族の関係を、標準的なジェノグラム記法（男性=□ / 女性=○ / 不明=◇、
故人=✕、本人=赤枠）で1枚に整理できる Web アプリです。
医療・介護のジェノグラムを起点に、表示を切り替えて相続関係図（銀行・士業）など
業種を問わず使えるツールサービスを目指しています。

## 特徴

- **必ず家系図として成立する自前レイアウトエンジン**
  - dagre 等の自動レイアウトに依存せず、世代を厳密に揃え、夫婦・きょうだいの
    サブツリーが決して重ならない決定的アルゴリズム（`lib/layout.ts`）。
- **再婚・離婚・複数婚・異母/異父きょうだいを表現できる union（夫婦）ベースのデータモデル**
- 同居グループ（枠で囲んで可視化）・既往歴・本人（プロバンド）設定
- FHIR R4（FamilyMemberHistory）入出力
- JSON 入出力（ロスレス保存）＋ localStorage 自動保存
- 和暦表示、検索、Undo/Redo、パン/ズーム

## 技術スタック

- Next.js 14 (App Router) / TypeScript
- 描画: 自前の SVG + HTML レンダラ（外部グラフライブラリ不使用）
- Tailwind CSS / lucide-react

## セットアップ

```bash
npm install
npm run dev    # http://localhost:3000
npm run build  # 本番ビルド
```

## 基本操作

- **追加**: ヘッダー右の「人物」ボタン、または人物を選択して出る
  ＋父 / ＋母 / ＋配 / ＋兄弟 / ＋子 ボタン
- **編集**: 記号をダブルクリック、または選択して「編集」
- **削除**: 編集ダイアログの「削除」、または選択して Delete キー
- **移動/拡大**: 背景ドラッグでパン、ホイールでズーム、⛶ で全体表示

## データ構造

```
types/genogram.ts   Person / Union / GenogramData
lib/layout.ts       レイアウトエンジン（世代計算＋再帰パッキング）
lib/genogramOps.ts  関係操作（親/配偶者/子/きょうだい追加・削除）＋関係線生成
lib/fhir.ts         FHIR R4 入出力
lib/storage.ts      自動保存 / JSON 入出力
components/Genogram.tsx     メイン（キャンバス・状態管理）
components/PersonSymbol.tsx ジェノグラム記号
components/EditDialog.tsx   人物編集
```

## 既知の制限 / 今後

- 同居グループの枠は構成員の外接矩形のため、枠内に非構成員が入る場合がある
- 姻族（婚入者）の祖先まで描く「2家系の結合」は簡易対応（別ルートとして並置）
- PDF / 画像 / CSV 出力は今回スコープ外（必要なら追加可能）
